# Introduction To Software Engineering Project

## Hospital Management System (HMS)

This repository contains all deliverables for the Software Engineering course project.

### Project Overview
The Hospital Management System (HMS) aims to streamline and automate hospital operations through a centralized, digital platform that integrates patient management, doctor scheduling, billing, and recordkeeping.

### Deliverable 1 Contents
- **System Vision Document**: Purpose, scope, and goals of HMS
- **Stakeholder Identification**: Operational, Executive, External, and Technical stakeholders
- **Functional Requirements**: Core system features and capabilities
- **Requirements Elicitation Techniques**: Interviews, Questionnaires, Observation, Document Analysis
- **Interview Agendas**: For Receptionist (Operational) and Hospital Administrator (Executive)
- **Questionnaire**: For operational users
- **Activity Diagrams**: 
  - Patient Appointment Booking Process
  - Billing and Discharge Process
- **Project Management**: Jira screenshots showing task tracking and timeline
- **Technical Report**: Complete PDF documentation
